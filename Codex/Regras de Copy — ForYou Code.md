# Regras de Copy — ForYou Code

> Feedback direto do Protagonista Ferreira (2026-05-04). Regras obrigatórias para todo copy da marca.

---

## ❌ Proibido

- **"Não é sobre X, é sobre Y"** — clichê de copywriting. Nunca usar.
- **Travessão (—)** — nunca usar em nenhuma peça.
- **Ruído** — sem enrolação, sem filler, sem palavras que não entregam valor.
- **Lacração** — sem discurso de ativismo, sem virtue signaling, sem pauta identitária.

---

## Princípios

- Direto ao ponto, sempre.
- Foco em resultado concreto: ROI, eficiência, ativo.
- Tom de autoridade, não de convencimento emocional.
