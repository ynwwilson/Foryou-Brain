# Rodrigo / Concretize Pré Moldados

## Negócio
Fábrica de pré-moldados (construção civil B2B).

## Status
Cliente ativo.

## Serviços contratados

| Serviço | Valor | Observação |
|---|---|---|
| Marketing mensal (Meta Ads + Google Ads + roteiros + social básico) | R$1.500/mês | Gasta menos de 2 dias/mês. Cobrado barato — novo padrão R$3.500-5k |
| IA de atendimento + CRM + painel completo | R$2.500 (único) | Preço promocional — valor correto R$12k-18k |

## Stack
Lovable + Supabase + WhatsApp API

## Por que a IA foi construída
Vendedor demorava para responder e perdia venda. Além disso, a ForYou Code ficava cega no marketing — sem dados das conversas de venda, não conseguia otimizar as campanhas. A IA fechou esse ciclo: leads qualificados automaticamente + dados reais alimentando o marketing.

## O que o sistema entrega (diferencial para usar como case)
- Painel completo com a marca do Rodrigo
- IA respondendo leads 24h com contexto do negócio
- Funil visual mostrando onde cada lead está
- Objeções mapeadas automaticamente
- Botão de pausar IA e assumir conversa com histórico completo
- Dados de conversa integrados com o marketing

## Uso como case
- Não divulgar o valor cobrado (R$2.500 foi promocional e errado)
- Usar o problema resolvido: "vendedor perdia lead por demora + marketing cego = construímos sistema que resolve os dois"
- Demo do painel é o principal ativo de venda para novos prospects de IA
