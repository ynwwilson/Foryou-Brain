# Como usar o Autentique para contratos

## O que é
Autentique é a ferramenta de assinatura digital usada pela ForYou Code para todos os contratos.

## Quando usar
- Todo projeto novo, independente do valor
- Qualquer adendo de escopo
- Contratos de mensalidade recorrente

## Cláusula de omissões
Todos os contratos da ForYou Code incluem cláusula de omissões customizada.
**O que ela faz:** define que o que não está explicitamente descrito no contrato não está incluído no escopo.
**Por que existe:** protege contra cliente que pede funcionalidades extras sem pagar.

## Fluxo
1. Redigir o contrato com escopo detalhado
2. Incluir cláusula de omissões
3. Enviar via Autentique para assinatura do cliente
4. Só iniciar o projeto após assinatura de ambas as partes

## Regra
Nunca começar trabalho sem contrato assinado.
"A gente se acerta depois" é como projetos viram prejuízo.
