# Processo de criação de conteúdo na ForYou Code

## Regra número 1
Nenhum conteúdo é produzido sem definir o funil antes.
TOF → MOF → BOF determina o hook, o desenvolvimento e o CTA.

## Fluxo de produção

### 1. Briefing
- Definir: qual sócio vai gravar? (Mestre, Marco ou Eduardo)
- Definir: qual etapa do funil? (TOF, MOF, BOF)
- Definir: qual formato? (vídeo curto, carrossel, story)

### 2. Roteiro
- Aplicar a estrutura de roteiro específica do sócio
- Mestre: 6 etapas, storytelling 1ª pessoa, sem moral explícita
- Marco: 5 etapas, espelho indireto, contraste de realidade
- Eduardo: 7 etapas, gancho provocativo, fechamento com resíduo

### 3. Revisão
- O roteiro está no funil correto?
- O gancho está nos primeiros 3 segundos?
- Tem CTA claro (para BOF) ou ausência intencional de CTA (para TOF)?

### 4. Gravação e edição
- Responsabilidade de cada sócio para seu próprio canal
- Identidade visual: cada perfil tem visual próprio e intransferível

### 5. Publicação e distribuição
- Meta Ads: impulsionar conteúdo BOF para lista de remarketing
- Orgânico: TOF e MOF para crescer audiência nova

## Marcas exclusivas do Mestre
- Doggs e Aurea são marcas do Mestre — invisíveis para os outros sócios
- Conteúdo dessas marcas segue estratégia separada
