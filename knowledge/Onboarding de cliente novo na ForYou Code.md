# Onboarding de cliente novo na ForYou Code

## O que fazer logo após o fechamento

1. **Contrato assinado** no Autentique antes de qualquer ação
2. **Coleta de ativos do cliente:**
   - Logo em SVG ou PNG fundo transparente
   - Paleta de cores (hex)
   - Domínio (ou definir novo)
   - Credenciais de acesso que serão necessárias
3. **Criar projeto no Supabase** com o nome do cliente
4. **Criar projeto no Lovable** e conectar ao Supabase
5. **Configurar domínio** no Cloudflare + Vercel
6. **Alinhar cronograma:** quais módulos em qual ordem

## O que o cliente precisa entender desde o início
- Ele terá acesso total ao sistema no final
- Mudanças fora do escopo geram novo contrato
- Prazo começa a contar após recebimento dos ativos

## Erros comuns a evitar
- Começar sem contrato assinado
- Começar sem receber logo e identidade visual
- Não definir quem aprova as entregas do lado do cliente
