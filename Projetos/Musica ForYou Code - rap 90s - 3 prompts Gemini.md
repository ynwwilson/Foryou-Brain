# Música ForYou Code — Rap Anos 90 (3 prompts de 30s)

> Gemini gera 30s por vez. Usar na ordem: Prompt 1 inicia, Prompt 2 e 3 continuam a mesma música.
> Música completa: 1min30 — intro + José (parte 1), hook + Marco (parte 2), Eduardo + final (parte 3).

---

## PROMPT 1 — Início (0:00–0:30)

Crie uma música de 30 segundos no estilo **rap anos 90 / boom bap clássico**:

**ESTILO / SOM:**
- BPM 90-92, bateria empoeirada estilo MPC, sample de jazz/soul ao fundo
- Estalo de vinil (vinyl crackle), baixo grave e quente, scratches de DJ na introdução
- Voz masculina grave, flow east coast anos 90, confiante e agressiva, sem autotune
- Vibe Racionais / Sabotage / Wu-Tang — cru, de rua, cabeça

**LETRA (cantar exatamente esta letra):**

[Intro — scratch de DJ, beat entrando]
Yo... ForYou Code no microfone
Três sócios, um plano, presta atenção no nome

[Verso 1 — José]
José no terminal, a IA é ferramenta
Pega tua bagunça e devolve sistema
Planilha, caderno, WhatsApp, improviso
Vira plataforma própria, código preciso
Duzentos aluno usando o que ele fez
Loja com PIX automático, mais de uma vez
Madrugada acesa, deploy todo dia
O Mestre não para — isso é tecnologia

---

## PROMPT 2 — Continuação (0:30–1:00)

**Continue exatamente a mesma música.** Mesmo beat boom bap anos 90, mesmo BPM 90-92, mesmo sample de jazz, mesma voz masculina grave com flow east coast. Não reinicie, não mude o instrumental. Esta é a parte 2 de 3.

**LETRA (cantar exatamente esta letra):**

[Refrão — vozes dobradas, scratch no fundo]
ForYou Code — sistema próprio na mão
Ferramenta pronta é remendo, não é solução
José constrói, Marco fecha o que tem
Eduardo viraliza — e a firma só vem

[Verso 2 — Marco]
Marco no telefone, a venda é xadrez
Objeção que aparece, ele quebra de vez
High-ticket na mesa, respeito no olhar
Não vende tela — vende a empresa decolar
O dono escutou, entendeu a missão
Contrato assinado com aperto de mão

---

## PROMPT 3 — Final (1:00–1:30)

**Continue exatamente a mesma música.** Mesmo beat boom bap anos 90, mesmo BPM 90-92, mesmo sample de jazz, mesma voz masculina grave com flow east coast. Não reinicie, não mude o instrumental. Esta é a parte 3 de 3 — termine a música com outro e fade out.

**LETRA (cantar exatamente esta letra):**

[Verso 3 — Eduardo]
Eduardo no corte, o meme é estratégia
Topo de funil, a internet contagia
Saiu de casa cedo, dezessete no mundo
Hoje o alcance dele chega em um segundo
A marca cresce enquanto o povo sorri
Quem riu da tropa hoje quer estar aqui

[Refrão final — vozes dobradas]
ForYou Code — sistema próprio na mão
Ferramenta pronta é remendo, não é solução
José constrói, Marco fecha o que tem
Eduardo viraliza — e a firma só vem

[Outro — beat desacelerando, scratch final]
De Patos de Minas pro mapa do Brasil
ForYou Code... a bagunça virou sistema
(fade out com vinil estalando)
