# Conteúdo ForYou Code (Leticia)

> Operação de conteúdo do **perfil institucional ForYou Code**, executada por [[Leticia]].
> Análogo do que [[Conteúdo José]] é para o canal pessoal de José.

## Função no funil
O perfil ForYou Code carrega o **TOF e BOF institucional**: gente conhece a marca pelo perfil, vê os cases, e cai no formulário de pré-briefing ([[Plano Comercial ForYou Code 2026]] §9). Os perfis dos sócios (José, Marco, Eduardo) carregam autoridade individual; o perfil da marca carrega *confiança institucional* + cases.

## Volume e distribuição (vindos do plano comercial)
- **6 vídeos/dia** = **42 vídeos/semana** = peça mais pesada do sistema de conteúdo
- **4 TOF** — notícias de IA/tech traduzidas pra dono de negócio
- **1 MOF** — dados de mercado aplicados
- **1 BOF / collab** — sócio aparece com produto + CTA

## Quem é o público (não confundir com o do José)
- Dono de clínica, escola, distribuidora, e-commerce, prestador de serviço
- Faturamento R$50k–500k+/mês
- **Não entende IA**, mas sabe que "tá perdendo onda"
- Quer ouvir "como isso afeta MEU negócio", não "como funciona por dentro"

## Tom dominante
- *Autêntico, real, direto. Fala como pessoa, não como empresa.*
- Tradução: amigo te contando a novidade, não jornal lendo manchete.
- Sem hype, sem coach, sem "absurdo/insano". Sem termo técnico cru.
