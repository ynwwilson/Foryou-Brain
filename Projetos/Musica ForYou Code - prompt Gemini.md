# Música ForYou Code — Prompt para Gemini

> Colar o bloco abaixo inteiro no Gemini (geração de música/song).

---

## PROMPT

Crie uma música no estilo **funk brasileiro / phonk agressivo (mandelão + Memphis phonk)** com estas características de produção:

**ESTILO / SOM:**
- BPM 132-140, batida de funk mandelão com cowbell distorcido de phonk brasileiro
- 808 pesado e arrastado (sliding bass), graves saturados e sujos
- Sintetizadores dark, atmosfera de guerra/dominação
- Vocal masculino grave, agressivo, estilo DJ de baile com autotune leve
- Ad-libs de DJ hypando ("HÃ!", "VAI!", "É A FORYOU!"), scratches e vocal chops
- Drop pesado no refrão, quebra de beat antes de cada verso
- Energia de entrada de lutador / motivação agressiva de quem está dominando o mercado
