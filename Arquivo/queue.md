# Inbox — Fila de Captura

> Jogue aqui qualquer coisa que aparecer: ideia, decisão, link, tarefa, insight.
> Não precisa estar organizado. Só não deixa acumular mais de 48h.

---

## Como processar

Para cada item abaixo:
- Se é um **fato permanente** → cria nota em `knowledge/`
- Se é uma **decisão de projeto** → vai para `atlas/`
- Se é uma **tarefa** → executa ou agenda
- Se é **lixo** → deleta sem culpa
