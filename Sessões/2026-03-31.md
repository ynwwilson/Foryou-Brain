## O que ainda precisa do Mestre (ações manuais)

### Prioridade 1 — Conectar OAuth (2 min cada)
1. Abrir Composio → auth-configs
2. Entrar em cada config
3. Clicar "Connect Account" → autorizar no browser

Começar por: **Gmail → Google Calendar → Slack → GitHub**

### Prioridade 2 — Instalar Pixel Agents
1. Abrir VS Code na pasta `claude-novo`
2. Pressionar F5
3. Na nova janela: Ctrl+Shift+P → "Pixel Agents: Abrir Sala de Controle"

### Prioridade 3 — Pendências que precisam de tokens
- Telegram: obter Bot Token do @BotFather
- Meta Ads / WhatsApp Business: configurar no Meta Developers
