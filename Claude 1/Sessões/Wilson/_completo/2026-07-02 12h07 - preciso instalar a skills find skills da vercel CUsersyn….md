Be cautious with anything under 100. 2. **Source reputation** — Official sources (`vercel-labs`, `anthropics`, `microsoft`) are more trustworthy than unknown authors. 3. **GitHub stars** — Check the source repository. A skill from a repo with <100 stars should be treated with skepticism.  ### Step 5: Present Options to the User  When you find relevant skills, present them to the user with:  1. The skill name and what it does 2. The install count and source 3. The install command they can run 4.
