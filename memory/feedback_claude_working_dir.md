Sempre rodar `claude` E `claude-nova` de dentro da pasta do vault (`Foryou-Brain`) — não do home directory.

**Why:** O CLAUDE.md está dentro do vault. Se rodar de fora, o contexto da ForYou Code não carrega.

**How to apply:** Ao orientar sobre uso do terminal, sempre indicar que o comando deve ser executado a partir de `C:\Users\eduar\Foryou-Brain` (Eduardo), `~/Foryou-Brain` (Marco) ou `C:\Users\ynwwi\Projects\claude-novo\stark\Stark` (Mestre). Vale para ambos os comandos: `claude` e `claude-nova`.
