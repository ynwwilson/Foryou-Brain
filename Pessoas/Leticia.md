# Leticia

## Info
- **Função:** Funcionária da ForYou Code. Grava vídeos para o **perfil institucional ForYou Code** (não canal pessoal) e, sob demanda, conteúdo para as empresas clientes.
- **Volume:** 6 vídeos/dia — peça mais pesada da máquina de conteúdo da ForYou Code.

## Distribuição diária no perfil ForYou Code
Definida no [[Plano Comercial ForYou Code 2026]] §8 "Sistema de Conteúdo":

| Quantidade | Tipo | Descrição |
|---|---|---|
| 4 | TOF | Notícias de IA/tech contextualizadas — *"como contando pra um amigo, não lendo jornal"* |
| 1 | MOF | Dados de mercado aplicados a negócios reais |
| 1 | BOF / collab | José ou Marco aparece mostrando produto, com CTA da marca |

## Tom obrigatório do perfil
> Autêntico, real, direto. Fala como pessoa, não como empresa. Nada formal, nada ensaiado.

Tradução: pega notícia de IA / mercado e explica para **dono de negócio que não entende IA** (clínica, escola, distribuidora, e-commerce) — não para builder/vibecoder (esse é o público do José).
